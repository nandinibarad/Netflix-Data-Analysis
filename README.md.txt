# 🎬 Netflix Data Analysis Project

##  Overview
This project analyzes a Netflix dataset using Python to discover trends in content type, release years, and ratings. It demonstrates data cleaning, exploratory data analysis, and visualization skills.

---

##  Objectives
- Analyze Movies vs TV Shows distribution
- Study release year trends
- Explore IMDb score distribution
- Generate visual charts automatically
- Practice real-world data analysis using Python

---

##  Tools & Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- VS Code

---




##  Visualizations Included
1. Movies vs Shows Count  
2. IMDb Score Distribution  
3. Top Release Years

---

##  How to Run Project

### 1️. Install Libraries

pip install pandas matplotlib seaborn

### 2️. Run Project

python src/analysis.py

---

##  Key Insights
- Movies are more common than TV Shows
- Certain years had higher content releases
- IMDb ratings mostly fall in mid-to-high range
